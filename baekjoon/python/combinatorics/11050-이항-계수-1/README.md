## Task
자연수 
N과 정수 
K가 주어졌을 때 이항 계수 
binomial{N}{K}를 구하는 프로그램을 작성하시오.
---

## Input Format
첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 10, 0 ≤ K ≤ N)

---

## Output Format
binomial{N}{K}를 출력한다.

---

## Sample Input 0
<pre>
5 2
</pre>

---

## Sample Output 0
<pre>
10
</pre>
